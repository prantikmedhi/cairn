Core Concepts & Features

Multi-Framework Execution

Execute your `.crn` loops natively on your preferred backend: `langchain`, `langgraph`, `crewai`, `autogen`, or `openai`.

View Adapters →

Sub-loop Composition

Break complex agents down into reusable modules by importing and composing loops from local files.

Checkpoint / Resume

Native support for saving and restoring long-running loop execution states. Pause execution and pick up right where you left off.

Read Docs →

Budget Guards & Circuit Breakers

Prevent runaway loops with hard caps on iterations, duration, and token cost. Prevent systemic failures with robust retry guards.

Parallel Fan-Out

Fork/join execution patterns for concurrent agent tasks. Execute independent tasks concurrently and join the results.

CairnStudio

Visual loop editor in active beta—bringing drag-and-drop loop engineering to your browser without needing to write raw YAML.

Try Beta →
