The Problem

What Is Agent Loop Engineering?

Agent loop engineering is the discipline of designing, validating, and orchestrating AI agent task loops — the sequential and parallel steps that define how an AI agent processes information, makes decisions, and takes actions. Every AI agent framework implements loops differently, creating vendor lock-in.

Cairn solves this with a vendor-agnostic DSL (.crn) that lets you define your agent loop once and run it on any framework.

Quick Start

Get up and running in seconds.

user@cairn: ~/project 
# Clone and setup
$ git clone https://github.com/prantikmedhi/cairn.git
$ cd cairn
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -e ".[dev]"

# Validate and run a loop
$ python3 -m cairn.main validate cairnlang/examples/hello-world.crn
$ python3 -m cairn.main run cairnlang/examples/hello-world.crn --input message=forge
