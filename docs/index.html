What Is Agent Loop Engineering?

Agent loop engineering is the discipline of designing, validating, and orchestrating AI agent task loops — the sequential and parallel steps that define how an AI agent processes information, makes decisions, and takes actions. Every AI agent framework (LangChain, LangGraph, CrewAI, AutoGen, OpenAI Agents SDK) implements loops differently, creating vendor lock-in.

Cairn solves this with a vendor-agnostic DSL (.crn format) that lets you define your agent loop once and run it on any framework. It's the write once, run anywhere for AI agent loops.

Loop Engineering in Action

A minimal agent loop in Cairn DSL:

# hello-world.crn — a minimal agent loop
loop:
  id: hello-world
  name: Hello World Loop
  version: "0.1.0"
  budget:
    max_iterations: 3
    max_duration: "30s"
  states:
    - id: greet
      handler: core.template
      inputs:
        prompt: "Hello, {{ message }}!"
      transitions:
        - done
    - id: done
      handler: core.collect

Supported Frameworks

Cairn's plugin architecture lets you target any agent framework. Built-in adapters:

CrewAI

AutoGen

OpenAI

Custom

Quick Start

git clone https://github.com/prantikmedhi/cairn.git
cd cairn
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
python3 -m cairn.main validate cairnlang/examples/hello-world.crn
python3 -m cairn.main run cairnlang/examples/hello-world.crn --input message=forge
